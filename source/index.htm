Portfolio

A selection of my works

Academis

Language: PHP, MySQL, JS

Academis is a school management information system comprising of numerous modules. It manages data for pupils, staff and guardians, records registration, gradebook, curriculum and behaviour management data, provides automated testing and marking and is used to write termly reports and generate invoices.
Euroharmony

Language: PHP, MySQL, JS

Euroharmony is a virtual airline management system. It manages pilot data for the VA and respective pilots. This includes recording pilot flight reports and details of the airline's fleet and routes. The system also interfaces with a flight logger application which runs on the desktop recording and reporting flight data.
ExamDisplay

Language: C#, WPF, XAML

ExamDisplay is a desktop application that aids with the delivery of public examinations at GCSE and A-Level. It presents all infomation required by the Joint Council for Qualifications in the UK on a selected screen, and allows for easy entry of required data. Dropdown lists content can be modified in file, as can default settings.
Digital Signage

Language: C#, WPF, XAML

Digital Signage is an application for displaying signage content on networked display screens. It periodically retrieves content from a central server location using Json, and then retrieves image and video content which are scheduled for display.
TwoTypes

Language: C#, WPF, XAML

Two types is a binary, denary and hex conversion game designed to to improve student ability in converting between these number systems. The game allows the user to select between different conversions and set the duration and difficulty of play.
Academis Tester

Language: (v1)Flex | C#, WPF

Academis Tester is the test delivery client for the tests held in Academis. This retrieves test and question data, presents the questions to the user and submits answers back to the server. It also includes a calculator, notepad, question navigation, timer and answered indicator.
